# Hell Let Loose Artillery Calculator

Simple artillery calculator for Hell Let Loose supporting:

## Supported Weapons

### SPA

-   Allies --- M4A3 (105mm)
-   Axis --- Sturmpanzer IV
-   Supports adjustment format (253+41, 500-21)

### Static Artillery

-   Allies & Axis
-   Range: 100m --- 1600m
-   Automatic mil conversion

------------------------------------------------------------------------

# How To Use

## 1. Launch

Double‑click:

HLL_Artillery_Calculator.exe

------------------------------------------------------------------------

## 2. Select Faction

Choose:

-   Allies (US)
-   Axis (German)

------------------------------------------------------------------------

## 3. Select Weapon

Choose:

-   SPA
-   Static Artillery

------------------------------------------------------------------------

# SPA Usage

Enter:

253\
253+41\
500-21

Example:

500+21

Calculator:

500m = 366mil\
366mil - 21mil\
Final = 345mil

Rules:

-   "+" subtracts mil
-   "-" adds mil

Max adjustable mil:

466 mil

------------------------------------------------------------------------

# Static Artillery Usage

Enter meters only:

100\
550\
1600

Example:

500

Calculator:

500m = 883mil

Range:

100m --- 1600m
